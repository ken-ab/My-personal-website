# Ken Zhang Personal Website

Personal portfolio website for Zhenkai Zhang (Ken), built with React, Vite, and TypeScript.

## Features

- Editorial-style personal homepage with profile card and portrait.
- Publications, internship and awards, development projects, and contact pages.
- Local data-driven content in `src/data/portfolio.ts`.
- Responsive layout for desktop and mobile viewing.
- No backend dependency.

## Tech Stack

- React
- Vite
- TypeScript
- React Router
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

The dev server runs at:

```text
http://localhost:5173
```

## Build

```bash
npm run lint
npm run build
```

## Content Editing

Most portfolio content lives in:

```text
src/data/portfolio.ts
```

The profile image lives at:

```text
src/assets/profile.jpg
```
